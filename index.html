üå≥ Cobb County Tree Preservation: Quick Facts

Effective standards for land development and construction (Section 416).

1. The Core Requirement: 15 Units Per Acre

Every developed site in Cobb County must maintain a minimum Site Density Factor (SDF) of 15 units per acre. This can be achieved by keeping existing trees, planting new ones, or a mix of both.

  • Residential Lots: Even in smaller subdivisions, every lot over 15,000 sq. ft. must have at least 2 units of trees.
  • The Goal: To control stormwater, reduce noise/glare, and ensure a healthy living environment.

2. Saving "Specimen Trees" (The VIPs)

The county grants special protection to "Specimen Trees"‚Äîtrees of significant size and health.

What qualifies?

  • Large Hardwoods: 30"+ diameter
  • Large Softwoods: 36"+ diameter
  • Understory Trees: 12"+ diameter
The Reward: Saving a Specimen Tree grants 3x the density credit.
The Penalty: Removing one requires high-value replacement (Section 416.5.3).

3. The "Protected Zone" & Critical Root Zone

The 12x Rule: The Critical Root Zone (CRZ) is a circle around the tree with a radius equal to 12 times the diameter of the trunk.

  • Strict Protection: No heavy equipment, soil storage, or chemical cleaning inside this zone.
  • Fencing: Orange safety fencing or chain link must be installed before any clearing begins.

4. Buffers & High-Value Forests

  • Zoning Buffers: Areas required to remain undisturbed by law.
  • High-Value Forest: Mature trees providing critical environmental functions (erosion control, screening). These are prioritized for preservation over new saplings.

üîç UDC Diagnosis: Tree Protection & Landscape

Summary of the November 2023 Code Assessment (II-71 to II-79)

Policy Gap

Specimen Tree "Administrative" Removal

Current rules allow Specimen Trees to be removed if "written documentation" is provided, but the code lacks specific criteria for the County to deny these requests.

The Recommendation: Require applicants to prove "Economic Hardship" or that site design is "Impractical" before a Specimen Tree can be cut.
Preservation

EDF vs. RDF: The Incentive Problem

While there is a "bonus" for keeping existing trees (EDF), stakeholders report that developers often clearcut sites and replace them with small saplings (RDF) because it is easier for construction.

Tree Type Existing (EDF) Unit Value Replacement (RDF) Unit Value
21" - 24" Tree 2.8 Units ‚Äî
4" Caliper Sapling ‚Äî 0.7 Units

It takes four 4-inch saplings to equal the "value" of one preserved 22-inch tree.

Technical Fix

Parking Lot Survivability

The diagnosis found a "gap" between parking space layout and tree survival. New standards suggest:

  • Mandatory Islands: One island and one canopy tree for every 10 contiguous parking spaces.
  • Shade Coverage: Overstory trees must be max 50 feet apart (trunk-to-trunk).
  • Root Space: Shift from "square footage" rules to minimum island dimensions (e.g., 9'x18').
New Standard

Defining "True" Open Space

Current code counts front/rear yards as "Open Space." The UDC update proposes a shift toward Common Open Space Set-Asides.

Type Includes
Natural Features Wetlands, flood hazards, steep slopes (>15%), and woodlands.
Passive Recreation Community gardens, gazebos, and formally planned greenery.
Active gathering Squares, plazas, forecourts, and civic greens.
Proposed Change

Street Tree Integration

Currently, trees in the public right-of-way do not count toward a project‚Äôs tree density (SDF). The update recommends allowing these to count to encourage "Complete Streets" and attractive streetscapes.
